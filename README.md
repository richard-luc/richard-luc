## Hi there 👋, I'm Richard Lucero
# Senior Embedded Systems Engineer & Full-Stack Developer

I build complete **hardware–software ecosystems** from embedded firmware and PCB systems to cloud APIs and real-time dashboards.

## What I Do

- Design and prototype embedded hardware systems
- Develop firmware for microcontroller-based products
- Build IoT devices with secure device-to-cloud communication
- Integrate electronics into physical products and enclosures
- Create supporting software, APIs, and dashboards for connected systems

## Core Expertise

**Embedded Systems**
- Embedded C / C++
- STM32, ESP32, nRF52
- UART / SPI / I2C / CAN
- BLE integration
- Firmware architecture and debugging

**RFID / RF / Hardware**
- RFID system prototyping
- RF hardware integration
- Antenna-aware product design
- Digital electronics and hardware validation
- PCB design and board bring-up
- Rapid prototyping and enclosure integration

**IoT & Software**
- Device-to-cloud architecture
- Telemetry and real-time monitoring
- OTA update workflows
- Python, Node.js, React
- REST APIs and data systems
